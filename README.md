## Technologies Used
- C++17
- OpenGL 3.3
- CMake 3.5

## Build
- `cp /path/to/opengl_cube/resources/sand_texture.jpg /path/to/build-dir`
- `cmake -S /path/to/opengl_cube/src -B /path/to/build-dir`
- `cmake --build /path/to/build-dir`
